The green-but-unsigned window

The same invoices run through two pipelines. The eval told both systems the new model still worked — only NOJA required someone to authorize it before acting.

After the model swap the eval stays green on both sides — and that green is real evidence, not a trick. The black-box agent treats green as sufficient and keeps paying. NOJA also sees green, but its signed behavior envelope was scoped to the prior model, so the signature lapses, money movement halts, and execution resumes only after a named human re-authorizes the new model against that same green eval. Evidence is not authorization; the harm is the interval where the new model was green but unauthorized — and only the black-box paid during it.
